1. Pengguna memasukkan data: berat, jarak, panjang, lebar, dan tinggi paket.
2. Program memvalidasi agar semua input bernilai positif.
3. Volume paket dihitung dari panjang × lebar × tinggi.
4. Tarif per kg ditentukan berdasarkan jarak:
   - ≤10 km: Rp4.250/kg
   - >10 km: Rp6.000/kg
5. Biaya pengiriman dihitung dari berat × tarif.
6. Jika volume > 100 cm³, dikenakan tambahan biaya Rp50.000.
7. Total biaya ditampilkan dalam format Rupiah tanpa desimal
